# 🐦 Django Tweet App

A full-featured Tweet Posting Web Application built using **Django** that allows users to register, login, create tweets with images, edit/delete their posts, and securely manage content.

---

##  Features

 User Registration & Login Authentication
 Create Tweet with Image Upload
 Edit and Delete Tweets (User-specific authorization)
 Tweet Listing with Latest First Order
 Secure Form Handling & Validation
 Media File Upload Support
 Clean Django MVT Architecture
 Responsive UI using Templates

---

## 🛠 Tech Stack

* **Backend:** Django (Python)
* **Database:** SQLite3
* **Frontend:** HTML, CSS, Bootstrap
* **Authentication:** Django Auth System
* **Media Handling:** Pillow

---

##  How to Run Locally

### 1️ Clone the Repository

```
git clone https://github.com/ashwani-codess/django-tweet-app.git
cd django-tweet-app
```

### 2️ Create Virtual Environment

```
python -m venv venv
venv\Scripts\activate
```

### 3️ Install Dependencies

```
pip install django pillow
```

### 4️ Run Migrations

```
python manage.py migrate
```

### 5️ Start Development Server

```
python manage.py runserver
```

Open in browser:
👉 http://127.0.0.1:8000/
